# Fan Profile Control

OpenDeck plugin for controlling Corsair fan speed profiles via [OpenLinkHub](https://github.com/jurkovic-nikola/OpenLinkHub) on [OpenDeck] (https://github.com/nekename/OpenDeck).

## Requirements

- OpenLinkHub running on `http://127.0.0.1:27003`
- Linux

## Actions

**Current Fan Profile** — Displays the active fan speed profile. Updates every 10 seconds. Press to refresh immediately.

**Set Fan Profile** — Opens a property inspector dropdown listing the available profiles (Quiet, Normal, Performance). Press the key to apply the selected profile.

## Installation

Install the `.streamDeckPlugin` file from `builds/` through the OpenDeck application.
